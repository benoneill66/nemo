What it does

A second memory that builds itself

It listens, transcribes, and every few minutes distills what mattered — categorized and cross-linked, like a knowledge graph of your day.

🎙️

Transcribes everything

Continuous on-device speech recognition. On macOS 26+ it uses Apple's SpeechAnalyzer enhanced-dictation engine for long-form accuracy and punctuation.

🧠

Builds memory

Distills recent transcript into durable notes via the Claude CLI — categorized into People, Projects, Decisions, Action Items, Ideas… and linked into a graph.

🗂️

Captures meetings

Say "start meeting" or hit a button. Everything said is grouped, and when it ends Claude writes a summary and folds it into your memory.

⭐

Mark by keyword

Say "important", "remember this", or "action item" and that moment is starred and pushed to high importance during consolidation.

🌱

Imports your context

Auto-discovers Claude's existing file memories and parses them directly — instantly, no LLM — preserving their categories and interconnections.

💬

Answers out loud

Say "Hey Nemo, …" and it routes the question to Claude and speaks the reply — while still transcribing in the background.

How it works

Mic to memory, all on your Mac

Nothing but distilled text ever leaves the device — and only to your own Claude CLI.

01

Listen

AVAudioEngine captures the mic in rolling windows.

02

Transcribe

Apple's Speech framework turns it into time-stamped text, on-device.

03

Distill

Every ~5 min the Claude CLI extracts durable, categorized memories.

04

Connect

Memories link by reference and shared entities into a graph.

05

Browse

A glassmorphic UI — plus plain JSON on disk you fully own.

Private by design

Raw audio never leaves your Mac

Speech is processed entirely on-device. Only distilled text is ever sent, and only to your own logged-in Claude CLI. No servers, no telemetry, no API keys — everything stored as plain JSON under ~/.config/nemo/ that you can read or delete anytime.

🔒 On-device transcription 🚫 No telemetry 🔑 No API keys 📄 Plain-JSON storage ⚖️ MIT licensed

Build it in under a minute

It's a single Swift package — clone, run one script, and allow the mic prompt. A locally built bundle isn't quarantined, so it opens with no Gatekeeper warning.
